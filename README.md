# 🚀 Telecom Customer Churn Prediction

## 📌 Overview
This project predicts whether a telecom customer is likely to churn using machine learning.

## 🔥 Features
- Real-time churn prediction
- Interactive SaaS-style dashboard
- Feature importance visualization
- Model explainability

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Render (deployment)

## 🌐 Live Demo
👉 [Click Here]((https://telecom-churn-predictor-mxx8.onrender.com))

## 📊 Model Details
- Algorithm: Random Forest
- Preprocessing: OneHotEncoder + StandardScaler (Pipeline)

## 💡 Key Insights
- Customers with month-to-month contracts have higher churn risk
- High monthly charges increase churn probability
